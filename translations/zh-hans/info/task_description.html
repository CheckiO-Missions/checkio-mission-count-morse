每个人和他们的兄弟肯定都听说过摩尔斯电码，这是一种将文字编码成二进制符号的经典编码方式，通常用点和破折号表示。然而，从技术上讲，这种编码是三元编码，因为两个字母之间的停顿实际上是一个单独的符号，这就好比你现在读到的单词之间的空白是字符，与那些有可见字形的字符是一样的。如果没有这些停顿来分隔单个的编码词，理论上，同一信息可以用指数级的多种不同方式解码。
