Кожен з нас напевно чув про азбуку Морзе - класичне кодування тексту двійковими символами, що канонічно позначаються крапками та тире. Однак це кодування технічно є потрійним, оскільки пауза між двома літерами насправді є окремим символом, аналогічно до того, як пробіли між словами, які ви зараз читаєте, є такими ж символами, як і ті, що мають видимі гліфи. Без пауз, що розділяють окремі кодові слова, одне й те саме повідомлення теоретично можна було б декодувати експоненціально багатьма різними способами.

Ваша функція отримує message без пауз разом з letters, кодування якого спочатку дало message. Ваша функція повинна підрахувати, скільки різних перестановок цих letters дають той самий message. Кожен окремий символ гарантовано з'являється у літерах не більше одного разу, всі символи і літери повинні бути використані.

Вхідні дані: Два аргументи: Повідомлення, закодоване азбукою Морзе, та послідовність літер у вигляді рядків (str{% if is_js %}ing{% endif %}).

Виведіть: Кількість перестановок як ціле число ({% if is_js %}number{% else %}int{% endif %}).
